Army Logistics & Operations Management System

About the project:
This project is a React-based web application developed to understand how logistics
and operations in the army can be managed digitally. The system shows how inventory,
soldiers, missions, and alerts can be handled using a single dashboard.


Features:

• Dashboard showing overall system status  
• Inventory management for weapons and equipment  
• Soldier profile management  
• Mission planning and tracking  
• Alerts and notification display  

Technologies Used:

• React.js  
• JavaScript  
• Tailwind CSS  

How to Run the Project:

1. Clone the repository  
2. Install dependencies using:
   npm install  
3. Start the application:
   npm start  
4. Open http://localhost:3000 in the browser

Note:

This project uses sample data for demonstration purposes and is not connected to a database.
